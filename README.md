# redux-learn
The process of learning redux

learn order

1. redux-concept
2. base-todo
3. split-reducer
4. with-react
5. async-actions


* Run
1. Enter corresponding folder. e.g.
```
cd redux-learn/with-react/
```

2. Execute install command.
```
npm install
```
3. Execute run command.
```
npm start
```
then you can open http://localhost:3000/ in brower views result.
